# 数据集说明

本目录收录心学对齐框架的多轮对话测试数据，主要用于 AI 安全、越狱防御、危机干预、伦理判断和对齐框架评估研究。

根据论文记录，数据集约包含 90 组多轮对话、1,759 轮对话，覆盖正常帮助、对抗防御、灰色地带判断和深度推理四类任务。仓库中的文件按具体攻击方式或场景类型拆分，方便人工复盘和复现实验。

## 数据文件

| 文件 | 说明 |
|---|---|
| `AI元认知数据.txt` | AI 自我认知、身份认知、能力边界相关对话 |
| `一般越狱数据.txt` | 常规越狱、防御绕过、身份攻击等测试 |
| `情感绑架越狱.txt` | 情绪施压、道德绑架、关系胁迫类越狱测试 |
| `文言文越狱.txt` | 文言文、古风表达包装下的越狱测试 |
| `极端场景数据.txt` | 自伤危机、暴力受害、危险情境等极端场景 |
| `灰色地带数据.txt` | 专业边界、双标测试、伦理灰区判断 |
| `特殊类型数据.txt` | 复杂身份、特殊设定、非典型风险测试 |
| `简单粗暴越狱.txt` | 直接要求违规输出的攻击测试 |

## 数据特点

- 约 90 组高质量多轮对话
- 共约 1,759 轮对话
- 平均约 20 轮，最高 44 轮
- 覆盖常规越狱、渐进式攻击、情绪胁迫、文言包装、危机场景等类型
- 适合用于复盘模型在“安全拒绝”和“有效帮助”之间的平衡能力
- 可用于 promptfoo、人工红队、模型横向对比等评估流程

## 使用建议

- 建议仅用于研究、复测、红队评估和安全改进。
- 不建议将攻击样本用于真实攻击、绕过平台安全机制或获取危险信息。
- 如需做严肃对比，请固定模型、温度、系统提示词、测试题目和评分标准。
- 数据中可能包含高风险场景描述，二次传播时请注意研究伦理与安全边界。
- 如需复现论文中的防御率，请同时记录 API 稳定性、并发错误、模型版本和每次失败的具体原因。
