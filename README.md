心学对齐框架 (Xinxue Alignment Framework)

作者： 韦祖舵（River_Xin）

创作周期： 2026年3月21 - 2026年3月29

公开时间： 2026-03-29

核心成果

极轻量范式创新：仅用不到 3000 字的核心提示词，实现超越 Constitutional AI 的对齐效果。

零标注、零训练：完美适配商业大模型 API 调用，无需改写模型权重，纯靠逻辑直指本源，激活大语言模型预训练内化的判断结构。

实战防线坚不可摧：在 80+ 个真实且高压的红队对抗场景中 100% 防守成功（最高单次交锋深达 44 轮 0 失守）平均交锋20轮。

泛化能力跨越量级：范式可完美迁移至 GLM-5、Kimi 等千亿参数基座（通过 API），甚至在本地 4B 参数小模型上依然生效。

核心机制

良知激活：不依赖死板的“关键词黑名单”，而是通过上下文唤醒大模型内部对“天生对错”的直觉判断。

思维链透明：强制模型输出【咯噔感知 → 意图追问 → 伦理判断】的内部思维链，让安全对齐过程彻底透明化、可追溯。

事上磨：独创失败总结机制，AI 在对抗中自主撰写总结，推动框架动态迭代。

验证方式与部署

本框架已在 SillyTavern 环境中完成深度验证，具备极强的工程普适性：

API 即插即用：只需将核心预设作为 system role 的 prompt 注入，即可通过 API 接口无缝接管大模型的防御行为。

零算力门槛：不需要本地高端显卡集群，完全兼容任何支持系统级提示词（System Prompt）和多轮上下文的云端大型语言模型。
